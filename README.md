# DR Portal - Disaster Recovery Replication Dashboard

A modern, responsive web-based mock portal for monitoring disaster recovery replication status across multiple cloud providers.

## Features

- **Multi-Cloud Support**: Monitor assets across AWS, Azure, and Google Cloud Platform
- **Real-Time Status**: Visual indicators showing replication health and progress
- **Asset Management**: Track VMs with standard naming conventions
- **Environment Tagging**: Differentiate between Production, Development, and Staging environments
- **Modern UI**: Clean, professional dashboard interface
